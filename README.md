# calctests

Simple project of functional tests for: http://www.calculator.net/conversion-calculator.html

## Instaling
```
   git clone  https://github.com/thiefik/calctests.git
```

## Running
To run all tests you need to triger run task from gradle. 
```
   ./gradlew test
```

## Used technologies
* Groovy 
* Gradle
* Spock
* Geb
